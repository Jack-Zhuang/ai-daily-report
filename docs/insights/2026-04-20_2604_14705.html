## 📖 背景与动机

人类活动轨迹（Human Activity Traces, HATs）是智慧城市、交通调度、行为预测等应用的核心数据基础。然而，真实采集的轨迹数据普遍存在两大问题：一是空间上呈现离散分布（如仅记录经过的POI而非连续路径），二是时间上不规则采样（GPS点间隔从秒级到小时级不等）。这些特性导致原始数据难以直接用于训练时空预测模型。传统方法通常依赖简单的插值或统计分布拟合来填补缺失值，但无法保证生成轨迹的时空连贯性和语义合理性。合成高质量的细粒度轨迹面临两个核心挑战：如何在空间上从离散点生成连续路径，以及如何在时间上从不规则采样恢复规则序列。SynHAT正是针对这两个挑战提出的创新解决方案。


**关键要点：**

- 真实HAT数据存在空间离散和时间不规则两大固有特性
- 现有方法难以生成既空间连续又时间规则的高质量轨迹
- 两阶段粗到细的策略可以有效解耦时空建模的复杂性

## 💡 核心方法

### 方法概述

SynHAT采用两阶段粗到细的扩散生成范式。第一阶段在粗粒度时空网格上生成离散事件序列，第二阶段通过细化机制将粗粒度表示逐步转换为细粒度连续轨迹。这种由粗到精的策略既保证了生成效率，又确保了轨迹的时空连贯性。


### 详细设计

在第一阶段（粗粒度生成），模型将时间轴划分为固定长度的时间槽（Time Slots），将空间划分为网格区域。每个时间槽内的活动被编码为离散的时空事件，扩散模型在这个粗粒度空间学习事件序列的分布。扩散过程从噪声开始，通过T步迭代逐步去噪，最终得到粗粒度的时空隐表示。这里T_i = [Int·i, Int·(i+1))表示第i个时间槽，整体时间范围T = {T_1, T_2, ..., T_D}。

第二阶段（细粒度细化）接收第一阶段的输出作为条件输入，逐步细化时间分辨率。具体来说，细化模型学习在给定粗粒度状态x_n的条件下，预测更细粒度的状态x_{n-1}。这个过程使用条件高斯分布建模：p_θ(x_{n-1}|x_n) = N(x_{n-1}; μ_θ(x_n, n), σ_θ(x_n, n)I)。通过多步细化，原本粗粒度的时间槽被逐步划分为更小的时间单元，最终生成精确到分钟甚至秒级的连续轨迹。

训练时采用去噪扩散模型的標準目标，通过最小化预测分布与真实后验分布的KL散度来优化模型参数。推理时，从纯噪声开始，两阶段级联执行，先得到粗粒度骨架，再逐步填充细节。

| 时间槽编码器 | 将不规则时间戳映射到规则时间槽网格，为后续扩散模型提供标准化的时序结构 |

import numpy as np
from typing import List, Tuple

# ============ 第一阶段：粗粒度扩散模型 ============

class CoarseGrainedDiffusion:
    """粗粒度时空事件生成器"""
    
    def __init__(self, time_slot_duration: int, grid_size: int, 
                 T_steps: int = 1000, feature_dim: int = 64):
        self.slot_duration = time_slot_duration      # 时间槽长度
        self.grid_size = grid_size                   # 空间网格大小
        self.T_steps = T_steps                       # 扩散步数
        self.feature_dim = feature_dim
    
    def create_time_slots(self, total_time: int) -> List[Tuple[int, int]]:
        """创建时间槽: T_i = [Int*i, Int*(i+1))"""
        slots = []
        for i in range(total_time // self.slot_duration):
            start = self.slot_duration * i
            end = self.slot_duration * (i + 1)
            slots.append((start, end))
        return slots
    
    def encode_to_grid(self, trajectory: np.ndarray) -> np.ndarray:
        """将轨迹编码为离散的时空事件网格"""
        # 伪代码：时空事件编码
        grid = np.zeros((self.grid_size, self.grid_size, len(self.create_time_slots(100))))
        return grid
    
    def forward_diffusion(self, x_0: np.ndarray, t: int) -> np.ndarray:
        """前向扩散：添加噪声"""
        noise = np.random.randn(*x_0.shape)
        alpha_bar = self.get_noise_schedule(t)  # 噪声调度
        return alpha_bar * x_0 + np.sqrt(1 - alpha_bar) * noise
    
    def get_noise_schedule(self, t: int) -> float:
        """噪声调度（简化的线性调度）"""
        return 1 - t / self.T_steps
    
    def denoise_step(self, x_t: np.ndarray, t: int) -> np.ndarray:
        """单步去噪（简化实现）"""
        # 实际使用U-Net或其他网络预测噪声
        predicted_noise = self.network_forward(x_t, t)  # 伪代码
        alpha = self.get_noise_schedule(t)
        # 简化去噪公式
        x_{t-1} = (x_t - (1-alpha) * predicted_noise) / np.sqrt(alpha)
        return x_{t-1}
    
    def network_forward(self, x: np.ndarray, t: int) -> np.ndarray:
        """简化的网络前向传播（伪代码）"""
        # 实际实现中应用Transformer/U-Net
        return np.random.randn(*x.shape) * 0.1
    
    def generate_coarse(self, num_slots: int) -> np.ndarray:
        """从纯噪声生成粗粒度时空表示"""
        x_T = np.random.randn(num_slots, self.feature_dim)
        
        # T步迭代去噪
        for t in range(self.T_steps, 0, -1):
            x_T = self.denoise_step(x_T, t)
        
        return x_T  # 粗粒度骨架


# ============ 第二阶段：细粒度细化模型 ============

class FineGrainedRefinement:
    """条件高斯细化器"""
    
    def __init__(self, refinement_steps: int = 4, feature_dim: int = 64):
        self.refinement_steps = refinement_steps  # 细化层级数
        self.feature_dim = feature_dim
    
    def split_time_slot(self, coarse_slot: np.ndarray, level: int) -> List[np.ndarray]:
        """将粗粒度时间槽划分为更细粒度"""
        # 每次细化将时间分辨率提高2倍
        num_fine_slots = 2 ** level
        fine_slots = np.array_split(coarse_slot, num_fine_slots)
        return fine_slots
    
    def refine_step(self, x_n: np.ndarray, n: int) -> np.ndarray:
        """
        单步细化：p_θ(x_{n-1}|x_n) = N(μ_θ(x_n, n), σ_θ(x_n, n)I)
        """
        # 预测均值和方差
        mu = self.predict_mean(x_n, n)
        sigma = self.predict_variance(x_n, n)
        
        # 条件高斯采样
        x_n_minus_1 = mu + sigma * np.random.randn(*mu.shape)
        return x_n_minus_1
    
    def predict_mean(self, x_n: np.ndarray, n: int) -> np.ndarray:
        """预测均值（伪代码）"""
        # 实际使用条件网络
        return x_n + np.random.randn(*x_n.shape) * 0.1
    
    def predict_variance(self, x_n: np.ndarray, n: int) -> np.ndarray:
        """预测方差（伪代码）"""
        return np.ones_like(x_n) * 0.5
    
    def cascade_refine(self, coarse_state: np.ndarray) -> np.ndarray:
        """级联细化：从粗粒度到细粒度"""
        current_state = coarse_state
        
        # 多层级逐步细化
        for level in range(1, self.refinement_steps + 1):
            current_state = self.refine_step(current_state, n=level)
            # 打印细化进度
            fine_resolution = 2 ** level
            print(f"  Level {level}: 分辨率细化到 {fine_resolution}x")
        
        return current_state  # 最终细粒度轨迹


# ============ 完整的两阶段模型 ============

class TwoStageTrajectoryGenerator:
    """两阶段时空轨迹生成模型"""
    
    def __init__(self, 
                 time_slot_duration: int = 3600,   # 1小时
                 grid_size: int = 100,
                 diffusion_steps: int = 100,
                 refinement_steps: int = 4):
        
        # 第一阶段：粗粒度
        self.coarse_model = CoarseGrainedDiffusion(
            time_slot_duration, grid_size, diffusion_steps
        )
        
        # 第二阶段：细粒度
        self.fine_model = FineGrainedRefinement(refinement_steps)
        
        self.time_slot_duration = time_slot_duration
    
    def train_step(self, real_trajectory: np.ndarray):
        """
        训练：最小化预测分布与真实后验的KL散度
        损失函数：去噪扩散目标
        """
        # 随机采样时间步
        t = np.random.randint(0, self.T_steps)
        
        # 添加噪声
        noisy_state = self.coarse_model.forward_diffusion(real_trajectory, t)
        
        # 预测噪声（实际训练时与网络预测比较）
        predicted_noise = self.coarse_model.network_forward(noisy_state, t)
        
        # 计算损失（简化）
        loss = np.mean((predicted_noise - 0) ** 2)  # 简化示例
        return loss
    
    def inference(self, total_time: int) -> np.ndarray:
        """
        推理：两阶段级联执行
        1. 从纯噪声生成粗粒度骨架
        2. 逐步填充细节
        """
        print("=" * 50)
        print("开始生成轨迹...")
        
        # --- 第一阶段：粗粒度生成 ---
        print("\n【第一阶段】粗粒度生成")
        num_slots = total_time // self.time_slot_duration
        coarse_state = self.coarse_model.generate_coarse(num_slots)
        print(f"  生成 {num_slots} 个粗粒度时间槽")
        
        # --- 第二阶段：细粒度细化 ---
        print("\n【第二阶段】细粒度细化")
        fine_trajectory = self.fine_model.cascade_refine(coarse_state)
        
        print("\n" + "=" * 50)
        print("轨迹生成完成！")
        
        return fine_trajectory

## ❓ 常见问题

**Q：为什么需要两阶段，直接在细粒度上生成不行吗？**

A：直接生成细粒度轨迹面临两个问题：一是高分辨率时空网格导致状态空间爆炸，扩散模型难以高效学习；二是细粒度的细节与粗粒度语义容易出现不一致。两阶段策略通过粗粒度阶段先建立语义骨架，再细化细节，既保证了效率又确保了语义一致性。


**Q：生成的轨迹能否直接用于训练下游预测模型？**

A：可以。论文的实验表明，使用SynHAT生成的轨迹数据训练的轨迹分类和位置预测模型，性能接近使用真实数据训练的模型。但需要注意，合成数据无法完全替代真实数据的多样性，建议在实际应用中进行数据混合使用。


**Q：与其他轨迹生成方法（如TimeGAN、TrajGAN）相比，SynHAT的优势在哪里？**

A：SynHAT的核心优势在于对时空不规则性的原生处理能力。传统GAN-based方法通常需要先将轨迹插值到规则网格，引入人为偏差。而扩散模型的迭代去噪过程更适合建模复杂的条件分布，能够更好地保持轨迹的时空统计特性。
