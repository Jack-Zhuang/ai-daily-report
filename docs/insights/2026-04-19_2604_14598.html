## 💡 核心方法

### 方法概述

CPGRec是一个三模块协同的推荐框架：准确性驱动模块负责基础推荐能力，多样性驱动模块通过流行度引导的重加权策略提升推荐广度，综合模块通过连接增强机制将两者有机融合。


### 详细设计

CPGRec的核心创新在于两个技术支柱。首先是**连接增强的邻居聚合（Connectivity-enhanced Neighbor Aggregation）**：传统图卷积网络在聚合邻居信息时，边的权重通常基于交互频率或相似度计算，CPGRec在此基础上引入游戏类别信息作为连接增强信号——同一类别的游戏之间即使交互稀疏，也会被赋予更高的聚合权重，这使得推荐系统能够发现具有潜在关联但交互数据稀少的游戏对。

其次是**流行度引导的边和节点重加权（Popularity-guided Edges and Nodes Reweighting）**：作者观察到长尾游戏中往往蕴含着高价值的个性化推荐机会。通过设计流行度感知的重加权机制，在信息传播过程中适度降低超热门节点的过度影响力，同时提升长尾节点的信息传递效率。具体而言，每层图卷积的聚合公式中引入可学习的重加权参数w_l和流行度调节因子，使得聚合过程既能保留高置信度的头部推荐，又能挖掘长尾区域的高质量推荐。

三个模块的协同方式为：准确性模块提供基础的交互预测能力，多样性模块通过重加权策略调整聚合过程中的信息流动，综合模块则在保持整体准确性的前提下，通过类别连接增强发现跨类别推荐机会。

```python
import numpy as np

# ============================================
# 简化的 CPGRec 推荐模型 - 伪代码实现
# ============================================

class CPGRec:
    def __init__(self, num_layers=2):
        self.L = num_layers
        # 可学习的重加权参数 (每层一个)
        self.w_l = [np.random.rand() for _ in range(self.L)]
    
    # ------------------------------
    # 模块1: 连接增强的邻居聚合
    # ------------------------------
    def connectivity_enhanced_aggregation(self, user, neighbors, edge_weights, 
                                            game_categories, category_similarity):
        """
        边的权重 = 交互权重 + 类别连接增强信号
        """
        enhanced_weights = []
        for neighbor in neighbors:
            # 基础交互权重
            base_weight = edge_weights[neighbor]
            # 类别连接增强: 同一类别游戏获得额外权重
            category_boost = category_similarity[user][neighbor] * 0.5
            # 综合权重
            enhanced_weights.append(base_weight + category_boost)
        
        # 归一化聚合
        return np.sum([n * w for n, w in zip(neighbors, enhanced_weights)])
    
    # ------------------------------
    # 模块2: 流行度引导的重加权
    # ------------------------------
    def popularity_guided_reweight(self, node_embeddings, popularity_scores):
        """
        降低超热门节点影响, 提升长尾节点信息传递效率
        """
        # 流行度调节因子 (可学习)
        alpha = 0.3  # 超参数
        reweighted = []
        for emb, pop in zip(node_embeddings, popularity_scores):
            # 流行度越低, 增益越高 (长尾提升)
            pop_factor = 1.0 / (1.0 + alpha * pop)
            reweighted.append(emb * pop_factor)
        return reweighted
    
    # ------------------------------
    # 模块3: 三模块协同
    # ------------------------------
    def forward(self, user, graph, game_info, popularity_scores):
        """
        准确性模块 + 多样性模块 + 综合模块
        """
        # --- 准确性模块: 基础交互预测 ---
        accuracy_score = self.base_prediction(user, graph)
        
        # --- 多样性模块: 流行度重加权 ---
        diversity_score = self.popularity_guided_reweight(
            graph.embeddings, popularity_scores
        )
        
        # --- 综合模块: 类别连接增强 + 层次聚合 ---
        final_embedding = self.layer_wise_aggregation(
            user, graph, game_info, popularity_scores
        )
        
        # 三模块融合
        combined_score = (0.5 * accuracy_score + 
                         0.2 * diversity_score + 
                         0.3 * final_embedding)
        
        return combined_score
    
    def base_prediction(self, user, graph):
        """准确性模块: 基础交互预测"""
        return np.dot(graph.user_emb[user], graph.item_emb)
    
    def layer_wise_aggregation(self, user, graph, game_info, popularity):
        """
        多层图卷积 + 连接增强 + 流行度重加权
        """
        embeddings = [graph.user_emb[user]]
        
        for layer in range(self.L):
            # 获取邻居信息
            neighbors = graph.get_neighbors(user)
            edge_weights = graph.get_edge_weights(user, neighbors)
            
            # 类别相似度矩阵
            cat_sim = self.compute_category_similarity(game_info)
            
            # 连接增强的邻居聚合
            aggregated = self.connectivity_enhanced_aggregation(
                user, neighbors, edge_weights, 
                game_info.categories, cat_sim
            )
            
            # 应用可学习的重加权参数 w_l
            reweighted = aggregated * self.w_l[layer]
            
            # 流行度调节
            pop_adjusted = reweighted / (1.0 + 0.1 * popularity)
            
            embeddings.append(pop_adjusted)
        
        return np.mean(embeddings, axis=0)
    
    def compute_category_similarity(self, game_info):
        """计算游戏类别相似度"""
        # 简化: 同一类别返回1, 否则返回0
        return lambda u, v: 1.0 if game_info[u].category == game_info[v].category else 0.0

    print("│   └── 边权重 += 类别相似度 * 增强系数")
    print("├── 流行度引导的重加权 (Popularity-guided Reweighting)")
```
