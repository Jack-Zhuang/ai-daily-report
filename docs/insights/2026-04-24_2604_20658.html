📖 背景与动机

随着大语言模型（LLM）能力的不断提升，基于多智能体协作的系统正被广泛应用于科学研究领域，包括数据分析、模型构建和科学报告生成等复杂工作流。然而，现实部署中的资源竞争（如GPU算力、API配额）要求智能体具备良好的协调与合作能力。传统的LLM评估主要关注单模型在特定任务上的性能，缺乏对多智能体协作能力的系统评估方法。行为经济学提供了丰富的博弈论框架，能够隔离和测量不同的合作机制（如信任、互惠、公平感知），这些机制在资源受限的多智能体场景中至关重要。本研究首次系统性地探索：单模型在行为经济学游戏中的表现，能否预测其在多智能体科学工作流中的团队协作效果。

关键要点：
  • 多智能体LLM系统在科研场景中的应用日益广泛，但缺乏有效的协作能力评估方法
  • 行为经济学博弈能够量化和隔离不同的合作机制，为评估提供理论基础
  • 现实科学工作流存在共享资源约束，合作能力直接影响系统效率

💡 核心方法

方法概述

本研究构建了「行为经济学游戏 → 合作特征提取 → AI科学任务验证」的完整评估pipeline。

详细设计

研究团队选取了六个经典行为经济学游戏，包括：公共物品博弈（Public Goods Game）、信任博弈（Trust Game）、囚徒困境（Prisoner's Dilemma）、最后通牒博弈（Ultimatum Game）、捐赠博弈（Dictator Game）和协调博弈（Coordination Game）。每个游戏设计多轮交互，记录35个开源LLM的策略选择、资源分配和合作行为。通过统计分析和机器学习方法，从六个维度刻画每个模型的合作特征：亲社会性、信任度、报复倾向、公平敏感度、风险偏好和协调能力。随后，将这些合作特征作为预测特征，输入到AI for Science任务中进行验证。任务包括：多智能体数据分析（协同处理多传感器异常检测）、多智能体建模（生态种群预测）和多智能体报告生成（协同撰写科学论文），所有任务均设置共享预算约束以模拟真实资源竞争场景。

"""公共物品博弈：贡献后乘以系数再均分"""

def play_trust_game(sender_amount, reciprocate_rate=0.5):

"""信任博弈：发送方投入，接收方返还"""

return received * reciprocate_rate

@staticmethod

def play_prisoners_dilemma(my_action, opponent_action):

"""囚徒困境：T>R>P>S"""

payoff_matrix = {

('D', 'C'): 5, ('D', 'D'): 1

return payoff_matrix[(my_action, opponent_action)]

def play_ultimatum(offer, responder_threshold=0.2):

"""最后通牒博弈：低于阈值则两败"""

if offer < responder_threshold:

return 0, 0

❓ 常见问题

Q：为什么用行为经济学游戏而非直接测试AI科学任务来评估模型？

A：行为经济学游戏具有高度标准化、可重复性强、计算成本低的优势，且能隔离出「合作」这一特定维度，避免任务复杂性的干扰。实验证明游戏表现与真实任务高度相关（r=0.73），因此可作为高效的预筛选工具。

Q：如果一个模型在游戏中表现「自私」但单模型能力很强，应该如何使用？

A：建议将其部署在需要强单兵能力的任务中（如特定领域的深度推理），同时避免让其主导需要资源协调的环节。更好的策略是在团队中配置1-2个高合作性模型作为「协调者」，配合若干专业模型实现优势互补。
