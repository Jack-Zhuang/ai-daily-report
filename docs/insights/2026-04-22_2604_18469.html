## 💡 核心方法

### 方法概述

广义综合控制方法（Generalized SCM）通过三步走策略实现动态基线估计：首先构建包含外生特征、滞后处理负荷和滞后供体信号的增强表示；其次利用正则化估计合成控制系数；最后基于优化目标生成动态counterfactual预测。


### 详细设计

经典SCM将处理单元Y_t建模为供体单元的加权组合Y_t = Σw_j × Y_j + ε_t，但存在两个核心缺陷：仅使用同期供体信息，忽略时间序列的自相关结构；残差被假设为不可预测的噪声。

本文的广义框架做出关键改进：

1. **增强表示矩阵D** = [J(外生特征) + M(滞后处理负荷) + L(选定滞后供体) + J(同期供体)]，维度为(J+M+L+J)

2. **优化目标**：min ||Y_t - D×w||² + R(θ)，其中R(θ)为正则化项

3. **动态预测能力**：通过引入滞后项，模型能够捕获负荷的自回归特性（AR structure），使残差从"不可预测噪声"变为"可建模的时间序列"

4. **供体选择机制**：不是使用全部供体，而是通过特征重要性或相关性筛选最相关的滞后供体信号，提高模型效率

该方法在保持SCM可解释性优势的同时，显著提升了预测精度。

| 组件 | 说明 |
|------|------|
| 合成控制权重估计器 | 基于优化目标min ||Y_t - D×w||² + λ||w||₁估计J+1维权重向量w，控制供体单元的贡献比例 |
| 动态特征增强模块 | 将外生特征(时刻/星期/季节)、滞后处理负荷(M阶)、选定滞后供体(L阶)拼接为增强矩阵D |
| 时变正则化机制 | 引入正则项R(θ)防止过拟合，允许权重随时间自适应调整，捕获非平稳性 |

### 💻 代码示例

```python
import numpy as np
from sklearn.linear_model import Ridge
from sklearn.feature_selection import SelectKBest, f_regression

# ============================================
# 广义SCM（增强合成控制法）- 简化实现
# ============================================

class GeneralizedSCM:
    """
    广义SCM框架：
    - 增强表示矩阵：外生特征 + 滞后处理负荷 + 选定滞后供体 + 同期供体
    - 带正则化的优化目标
    - 动态预测能力
    """
    
    def __init__(self, alpha=1.0, n_lags=3, n_selected_donors=5):
        self.alpha = alpha  # 正则化系数
        self.n_lags = n_lags  # 滞后阶数
        self.n_selected_donors = n_selected_donors  # 选定的滞后供体数
        self.weights = None
        self.donor_selector = None
        
    def _build_enhanced_matrix(self, Y_donor, Y_treated, X_exog):
        """
        构建增强表示矩阵 D
        D = [J(外生特征) + M(滞后处理负荷) + L(选定滞后供体) + J(同期供体)]
        """
        T, J = Y_donor.shape  # 时间步数、供体单元数
        
        # ① 外生特征矩阵 J
        J_matrix = X_exog
        
        # ② 滞后处理负荷 M (捕获自回归结构)
        M_matrix = np.column_stack([
            Y_treated[max(0, t-self.n_lags):t] 
            for t in range(T)
        ])
        
        # ③ 同期供体矩阵 J
        J_contemporary = Y_donor
        
        # 合并矩阵
        D = np.column_stack([J_matrix, M_matrix, J_contemporary])
        
        return D
    
    def _select_lagged_donors(self, Y_donor, Y_treated):
        """
        供体选择机制：筛选最相关的滞后供体信号
        使用特征重要性或相关性筛选
        """
        # 创建滞后供体候选矩阵（简化：使用各供体的滞后值）
        lagged_features = []
        for j in range(Y_donor.shape[1]):
            for lag in range(1, self.n_lags + 1):
                lagged_features.append(np.roll(Y_donor[:, j], lag))
        lagged_candidates = np.column_stack(lagged_features)
        
        # 选择最相关的K个滞后供体
        selector = SelectKBest(f_regression, k=self.n_selected_donors)
        selected = selector.fit_transform(lagged_candidates, Y_treated)
        
        return selected, selector
    
    def fit(self, Y_donor, Y_treated, X_exog):
        """
        拟合模型：min ||Y_t - D×w||² + α||w||²
        """
        # Step 1: 构建增强矩阵
        D = self._build_enhanced_matrix(Y_donor, Y_treated, X_exog)
        
        # Step 2: 供体选择
        L_matrix, self.donor_selector = self._select_lagged_donors(Y_donor, Y_treated)
        
        # Step 3: 合并最终特征矩阵
        D_final = np.column_stack([D, L_matrix])
        
        # Step 4: 带正则化的最小二乘
        model = Ridge(alpha=self.alpha)
        model.fit(D_final, Y_treated)
        
        self.weights = model.coef_
        self.feature_names = self._get_feature_names()
        
        return self
    
    def predict(self, Y_donor_test, Y_treated_lag, X_exog_test):
        """
        动态预测：利用AR结构和选定的滞后供体
        """
        # 重构增强矩阵
        D = self._build_enhanced_matrix(Y_donor_test, Y_treated_lag, X_exog_test)
        L_test = self.donor_selector.transform(
            self._create_lagged_donor_candidates(Y_donor_test)
        )
        D_final = np.column_stack([D, L_test])
        
        return D_final @ self.weights
    
    def _get_feature_names(self):
        """特征名称：用于可解释性"""
        return ['exog', 'treated_lag', 'donor_contemporary', 'donor_selected_lag']


# ============================================
# 使用示例（伪数据）
# ============================================

if __name__ == "__main__":
    # 模拟数据
    np.random.seed(42)
    n_time = 50
    n_donors = 6
    n_exog = 3
    
    Y_donor = np.random.randn(n_time, n_donors)       # 供体单元负荷
    Y_treated = np.random.randn(n_time)               # 处理单元负荷
    X_exog = np.random.randn(n_time, n_exog)         # 外生特征
    
    # 划分训练/预测期
    pre_treatment = 30
    
    # 训练模型
    gscm = GeneralizedSCM(alpha=1.0, n_lags=3, n_selected_donors=5)
    gscm.fit(Y_donor[:pre_treatment], Y_treated[:pre_treatment], X_exog[:pre_treatment])
    
    # 预测（动态）
    Y_pred = gscm.predict(
        Y_donor[pre_treatment:],
        Y_treated[pre_treatment-1:],  # 使用上一期作为滞后
        X_exog[pre_treatment:]
    )
    
    print(f"预测完成 | 权重维度: {gscm.weights.shape}")
    print(f"核心优势: 捕获AR结构 + 供体筛选 + 可解释性")
```

## ❓ 常见问题

**Q：广义SCM与经典SCM的核心区别是什么？**

A：经典SCM是静态的，仅用同期供体加权；广义SCM引入滞后项，将残差从"白噪声"变为可建模的AR过程，实现动态counterfactual预测


**Q：为什么需要筛选滞后供体而不是使用全部？**

A：全部使用会引入过多噪声和维度灾难。筛选机制（如基于相关性的前向选择）能够保留最相关的历史信息，提高模型信噪比和泛化能力


**Q：该方法在工业落地时最大的障碍是什么？**

A：数据门槛：需要足够多的相似建筑作为供体池。建议初期在建筑密集的工业园区或高校宿舍区试点，积累足够样本后再推广
