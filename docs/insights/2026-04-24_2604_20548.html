方法概述

该方法提出一种多智能体迭代规划框架，多个专业化智能体分别负责不同任务（如文献分析、创意生成、方案评估），通过迭代搜索和反馈机制逐步优化科研创意的质量和多样性。

详细设计

框架包含三个核心阶段：组合生成、迭代评估和方案优化。在组合生成阶段，系统从已有的研究语料库中提取关键元素（如研究问题、方法、数据集、评估指标），通过跨领域组合生成初步创意候选。在迭代评估阶段，每个智能体对其生成的创意进行多维度评分（新颖性、可行性、影响力等），同时计算创意之间的相似度以保证多样性。在方案优化阶段，系统根据评分和相似度反馈，指示智能体调整或扩展其创意方向。整个过程迭代进行，直到生成的创意满足质量阈值或达到预设的迭代次数。特别地，框架引入了「去重机制」——通过余弦相似度(sim(a_i, r_j))识别并淘汰高度相似的创意，确保输出结果的多样性。

"""组合生成阶段：从语料库提取元素，生成创意候选"""

# 随机组合关键元素（研究问题+方法+数据集等）

def calculate_similarity(idea1: Idea, idea2: Idea) -> float:

"""计算两个创意的余弦相似度（伪实现）"""

# 实际应使用embedding向量计算

shared = len(set(idea1.elements) & set(idea2.elements))

return shared / max(len(idea1.elements), len(idea2.elements), 1)

def evaluate_idea(idea: Idea) -> Dict[str, float]:

"""迭代评估阶段：多维度评分"""

return {

"新颖性": random.uniform(0.5, 1.0),

"可行性": random.uniform(0.5, 1.0),

"影响力": random.uniform(0.5, 1.0)

}

def deduplicate(ideas: List[Idea], threshold: float = 0.6) -> List[Idea]:

"""去重机制：淘汰高度相似的创意"""

filtered = []

for new_idea in ideas:

is_duplicate = False

for existing in filtered:

if calculate_similarity(new_idea, existing) >= threshold:

is_duplicate = True

break

if not is_duplicate:

filtered.append(new_idea)

return filtered

def optimize_ideas(ideas: List[Idea]) -> List[Idea]:

"""方案优化阶段：根据反馈调整/扩展创意"""

optimized = []

for idea in ideas:

# 根据评分决定是否扩展元素

if random.random() > 0.5:

new_elements = idea.elements + [random.choice(CORPUS)]

optimized.append(Idea(new_elements))

else:

optimized.append(idea)

return optimized

def is_quality_satisfied(ideas: List[Idea]) -> bool:

"""检查质量阈值"""
