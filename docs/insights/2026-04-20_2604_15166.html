## 📖 背景与动机

随着深度学习模型在敏感场景的广泛应用，机器遗忘（Machine Unlearning）技术应运而生，旨在满足用户数据删除权和模型更新需求。类遗忘（Class Unlearning）作为其重要分支，要求模型在移除特定类知识的同时保留其他类的性能。然而，现有方法普遍存在“虚假遗忘”问题：虽然分类器对遗忘类的输出置信度降低，但遗忘信息仍以隐藏表示形式存在于网络深层。研究者发现这主要源于两个机制——分类器头抑制与表示层级解耦的不一致——导致现有方法呈现弱/负选择性、深层表示结构残留、以及对偏置项的过度依赖。


**关键要点：**

- 类遗忘的核心挑战：降低分类准确率≠真正移除知识，表示层可能仍保留遗忘信息
- 现有方法的通病：分类器头抑制掩盖了深层表示中遗忘知识的持续存在
- 研究动机：需要一种能够在表示层级而非仅在输出层实现遗忘的方法

## 💡 核心方法

### 方法概述

DAMP是一种一次性闭式权重手术方法，针对预训练网络的每一层可学习算子，在其输入空间计算类原型，提取遗忘方向作为相对于保留类原型的残差，并通过投影矩阵移除这些遗忘方向。


### 详细设计

DAMP的核心思想是将类遗忘问题形式化为在表示空间中的方向移除问题。对于网络中的每个可学习算子（如线性层、卷积层），方法首先在算子的输入空间计算各类样本的原型表示。具体地，对每个遗忘类f，计算其在输入空间中的类中心，同时对保留类计算聚类中心。接着，将遗忘类的原型表示投影到由保留类原型张成的子空间的正交补上，得到遗忘方向向量。这些方向向量编码了遗忘类区别于保留类的独特信息。

在得到遗忘方向后，方法构建投影矩阵P_k，该矩阵能够将任意输入向量投影到遗忘方向的零空间，从而实现对遗忘信息的过滤。对于每个算子，其权重矩阵W被替换为P_k·W_new的形式，确保后续前向传播中遗忘信息不再被传递。

关键创新点在于“深度感知”特性：不同于仅在最终层操作的方法，DAMP逐层处理，在网络深层能够更彻底地移除遗忘知识，因为遗忘信息在深层往往更隐蔽地编码于表示中。通过递归地处理所有可学习算子，方法确保遗忘知识在整个网络中的一致性移除。

```python
import numpy as np

# 简化版 DAMP 类遗忘方法演示

class SimplifiedDAMP:
    """DAMP简化实现 - 演示核心思想"""
    
    def __init__(self, model_weights, class_list):
        """
        参数:
            model_weights: 模拟的网络权重列表 [W1, W2, W3, ...]
            class_list: 所有类别列表，格式如 ['forget_class', 'retain_class1', 'retain_class2']
        """
        self.weights = model_weights
        self.forget_class = class_list[0]
        self.retain_classes = class_list[1:]
        
    def compute_class_prototypes(self, X, y, class_name):
        """计算某个类在输入空间中的原型表示（类中心）"""
        class_mask = (y == class_name)
        class_samples = X[class_mask]
        prototype = np.mean(class_samples, axis=0)  # 类中心
        return prototype
    
    def compute_projection_matrix(self, retain_prototypes):
        """
        构建投影矩阵P_k
        将向量投影到遗忘方向的正交补空间（即零空间）
        """
        # 将保留类原型组成矩阵
        C = np.stack(retain_prototypes, axis=0)  # shape: (num_retain, feature_dim)
        
        # 计算保留类张成的子空间
        # 使用SVD分解获取正交基
        U, S, Vt = np.linalg.svd(C, full_matrices=False)
        
        # 投影矩阵 P = I - UU^T
        # 将向量投影到U的正交补空间
        I = np.eye(C.shape[1])
        P = I - U @ U.T
        
        return P
    
    def compute_forget_direction(self, forget_proto, retain_prototypes):
        """
        计算遗忘方向向量
        将遗忘类原型投影到保留类子空间的正交补上
        """
        C = np.stack(retain_prototypes, axis=0)
        U, _, _ = np.linalg.svd(C, full_matrices=False)
        
        # 正交补投影
        forget_direction = forget_proto - (U @ U.T) @ forget_proto
        return forget_direction
    
    def project_weight(self, P, W):
        """
        修改权重: W_new = P_k · W
        实现对遗忘信息的过滤
        """
        return P @ W
    
    def forgetting_process(self, X_train, y_train):
        """
        逐层遗忘过程（深度感知）
        """
        new_weights = []
        
        for layer_idx, W in enumerate(self.weights):
            print(f"\n{'='*50}")
            print(f"处理第 {layer_idx + 1} 层 (深度感知)")
            
            # Step 1: 计算各类原型
            forget_proto = self.compute_class_prototypes(X_train, y_train, self.forget_class)
            retain_prototypes = [
                self.compute_class_prototypes(X_train, y_train, c) 
                for c in self.retain_classes
            ]
            
            print(f"  遗忘类原型维度: {forget_proto.shape}")
            print(f"  保留类原型数量: {len(retain_prototypes)}")
            
            # Step 2: 计算遗忘方向
            forget_direction = self.compute_forget_direction(
                forget_proto, retain_prototypes
            )
            print(f"  遗忘方向向量范数: {np.linalg.norm(forget_direction):.4f}")
            
            # Step 3: 构建投影矩阵
            P = self.compute_projection_matrix(retain_prototypes)
            print(f"  投影矩阵维度: {P.shape}")
            
            # Step 4: 投影权重（关键步骤）
            W_new = self.project_weight(P, W)
            new_weights.append(W_new)
            print(f"  权重更新: {W.shape} -> {W_new.shape}")
            
            # 深层处理：更彻底移除（模拟）
            if layer_idx >= 2:
                print(f"  [深层增强] 在深层进行更彻底的遗忘知识移除")
        
        return new_weights
    
    def run(self, X_train, y_train):
        """运行完整遗忘流程"""
        print("DAMP 类遗忘方法开始执行")
        print(f"遗忘类: {self.forget_class}")
        print(f"保留类: {self.retain_classes}")
        
        # 执行逐层遗忘
        new_weights = self.forgetting_process(X_train, y_train)
        
        print("\n" + "="*50)
        print("遗忘完成！遗忘信息已被过滤。")
        return new_weights


# ==================== 使用示例 ====================
if __name__ == "__main__":
    
    # 模拟数据

    n_samples = 100
```

## ❓ 常见问题

**Q：DAMP与现有的梯度上升类方法（如SGU）有何本质区别？**

A：核心区别在于操作层级和优化方式。SGU等方法通过梯度上升降低遗忘类样本的损失来间接影响表示，需要多轮迭代优化；DAMP则直接在权重层面进行闭式投影手术，一次性完成表示修改。前者是隐式优化，后者是显式移除。


**Q：如何评估类遗忘的“彻底性”？现有指标有何局限？**

A：论文指出仅用分类准确率评估存在局限，因为分类器头可被抑制而表示仍保留知识。论文建议使用表示距离度量（RDM）和选择性指标，结合激活可视化等多维度评估。DAMP的实验正是通过这些指标验证了其在深层表示层面的遗忘有效性。
